# 前言

# 项目介绍

搭建一个自己的多功能云盘。
如阿里云盘 (opens new window)，百度云盘 (opens new window)那样。同时，可以让您快速搭建起公私兼备的网盘系统。 在底层支持不同的云存储平台，用户在实际使用时无须关心物理存储方式。可以运用于，个人用网盘、文件分享系统，亦或是针对大小团体的公有云系统和私密的加用NAS云。

# 实现方案

所选 作用 优点 说明
1.网盘程序 Cloudreve (opens new window) 提供核心服务 ... 方案依托（必选）
2.离线下载 Aria2 (opens new window) 对接网盘离线下载服务 ... 按需选择（可选）
3.底层服务 Docker (opens new window) 方便快捷且安全的运行上层服务 使用方便，部署快捷，可自动选择适应当前依托硬件的ARM,X86，等等架构。 方案依托（必选）
4.内网穿透 网云穿 (opens new window) 将网盘映射出去实非内网访问 据作者测试，他免费提供一条隧道，3M的带宽，且付有免费二级域名 按需选择（可选）
5.反向代理 NGINX (opens new window) 获得更丰富的WEB服务的扩展功能 ... 按需选择（可选）
6.运维面板 宝塔面板 (opens new window) 提供方便的图形运维界面 对于小白来说真是方便 方案依托（必选）
7.操做系统 linux (opens new window) 基础 ... 方案依托（必选）
8.依托硬件 树莓派3B (opens new window) 基础 作者手中闲置的一块 方案依托（必选）

注意

  • 上述实现方案的“说明”仅仅是基于本笔记的实际操作来定的，当然可在同类产品中找到替代的，就比如说，树莓派3B+或4B同样可以适用于本笔记或使用Linux的云服务器同样可以，按实际情况小心选择替代。
  • 如果你有能力完全可以按照方案来更换部分“所选”，就比如依托硬件可以说群晖NAS，那么“7.操作系统”和“6.宝塔面板”完全可以去除，使用群晖自带的套件去实现。当然你需要有一定的基础能力才行。
  • 本笔记只是众多方案其中之一，是作者认为更适用于大多数“极客玩家”的同时又兼容了实用性，安全性，以及可玩性的最简单的方案。
  • 作者强烈建议，如果再有能力的情况下请结合实际，做出最适用于自己的方案。
